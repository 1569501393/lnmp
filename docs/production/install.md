# 生产环境

请首先在 [`开发环境`](../development.md) 中熟悉本项目。

## 使用一键安装脚本

```bash
$ curl -fsSL lnmp.khs1994.com -o lnmp.sh ; sh lnmp.sh production
```

## 使用 `git clone`

```bash
$ git clone --depth=1 git@github.com:khs1994-docker/lnmp.git
```

## 更新

```bash
$ ./lnmp-docker.sh update

$ ./lnmp-docker.sh restart
```
